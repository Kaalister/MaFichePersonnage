Bienvenue dans la page d'édition, ici vous pouvez définir les caractéristiques BASIQUES de votre personnage.
Description :
Caractéristiques : Correspondent aux stats principales qui ne peuvent pas dépasser une valeur de base (PV/MANA/NIVEAU/...)
Compétences : Correspondent aux capasités de votre personnage
Stats : Correspondent aux states secondaires de votre personnage
Utilisation :
Pour supprimer une stat/compétence/caratéristique, videz le/les champ(s) associé(s).
Pour ajouter une stat/compétence/caratéristique, remplir les champs d'ajout associés et cliquez sur le +
Pour modifier un élément, modifiez simplement l'élément.

Pour revenir à l'écran de jeu, cliquez sur le bouton retour.
