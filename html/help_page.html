Bienvenue dans la page d'édition, ici vous pouvez définir les caractéristique BASIQUES de votre personnage.
Description :
Caractéristiques : Correspondent aux stats principales qui ne peuvent pas dépasser une valeur de base (PV/MANA/NIVEAU/...)
Compétences : Correspondent aux capasitées de votre personnage
Stats : Correspondent aux secondaires de votre personnage
Utilisation :
Pour supprimer des stat/compétence/caratéristique, videz les champs à effacer .
Pour ajouter une stat/compétence/caratéristique, remplir les champs associés et cliquez sur le +
Pour modifier un element, simplement modifiez l'élément.

Pour revenir à l'écran de jeu cliquez sur le bouton retour.
